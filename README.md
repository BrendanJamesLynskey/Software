## Cloud Service Models (`*aaS`)

A six-deck consolidated treatment of cloud service models — IaaS / PaaS / SaaS / CaaS / FaaS / AIaaS — with deep coverage of architecture, deployment, security and the LLM-as-a-Service landscape. Companion to the Deployment, Docker and Monetising decks above; new content is the architectural and operational depth (multi-tenancy, IAM, compliance, managed AI) those decks deliberately don't cover.
